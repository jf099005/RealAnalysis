## Theorem
if $f\ge 0$ on $E$, then $R(f,E)$ is measurable in $\mathbb R^{n+1}\implies f$ is a measurable function.

proof.
it suffices to that $\forall y\in\mathbb R$, the set $\{x:f(x)>y\}$ is measurable.

For $y \le 0$, then $\{f(x)>y\}=E$ is measurable

Now for $y>0$, $\{(x,y): f(x)>y,0\le y < \infty\}=R(f,E)$  is measurable

Let $F(x,y)=\chi_{\mathbb R(f, E)}$ is a  measurable function,

Hence by Tonelli's theorem $\forall$ almost everywhere $y\in [0,\infty)$, $F(x,y)$ is measurable in $\mathbb R^n$
$\implies \{f(x)\ge y\}$ is measurable

Therefore, $\{f(x)\ge y\}$ is measurable for a dense set of $[0,\infty)$

Hence, $f$ is measurable.

## Marcinkiewicz Theorem
Let $F$ be a closed set in a bounded open interval $(a,b)$, Define $\delta(x)=\delta(x,F)=\inf \{|x-y|:y\in F\}$
Then $\forall \lambda > 0$, the integral $M_\lambda (x)=\int_a^b \frac{\delta(y)^\lambda}{|x-y|^{\lambda+1}}$ is finite a.e. in $F$

Moreover, $M_\lambda\in L(F)$ and $\int_F M_\lambda (x)\le 2\lambda^{-1} |G|, G=(a,b)-F$

#### Remark
1. $\delta(x)$ is 1-Lipchitz function, i.e. $|\delta(x)-\delta(y)|\le |x-y|$
2. $\delta(x)=0\forall x\in F$
3. if $x\notin F$, $M_\lambda (x)=\infty$

Assume $x\in F$, then $\delta(x)=0$

Hence, $\frac{\delta^\lambda (y)}{|x-y|^{\lambda+1}}=\frac{[\delta(y)-\delta(x)]^\lambda}{|x-y|^{\lambda+1}}\le \frac{|x-y|^{\lambda}}{|x-y|^{\lambda+1}}=\frac 1{|x-y|}$

But $\int_a^b \frac{1}{|x-y|}dy=\infty$ (we want finite a.e. in $F$)

##### proof of Marcinkiewicz

By definition, 
$M_\lambda(x)=\int_a^b \frac{ \delta(y^\lambda) }{|x-y|^{\lambda+1}}dy=\int_G\frac{ \delta(y^\lambda) }{|x-y|^{\lambda+1}}dy$ (since $\delta(y)=0$ in $F$)

$\int_F M_\lambda(x)dx=\int_F (\int_G \frac{\delta(y)^\lambda}{|x-y|^{\lambda+1}}dy)dx\stackrel{Tonelli's Thm}{=} 
\int_G (\int_F \frac{ \delta(y^\lambda) }{|x-y|^{\lambda+1}}dx)dy$
$=\int_G\delta(y)^\lambda (\int_F \frac 1{|x-y|^{1+\lambda}}dx)dy$

observe that $y\in G$ and $x\in F$ we have
$|x-y| \ge \delta(y)-\delta(x) \ge \delta(y)$ ($\delta(x)=0$)

So $\int_F \frac 1{|x-y|^{1+\lambda}}dx \le \int_{|x-y| \ge \delta(y)} \frac 1{|x-y|^{1+\lambda}}dx \le \int_{\delta(y)}^{\infty} \frac{dt}{t^{1+\lambda}}=2\lambda^{-1} \delta(y)^{-\lambda}$

Then $\int_F M_\lambda(x)dx \le \int_G \delta^\lambda (y)[2\lambda^{-1}\delta(y)^{-\lambda}]dy = 2\lambda^{-1}|G|<\infty$

So, $M_\lambda(x)\in L(F)$ and is finite a.e. in $F$.
