# EX1.
fix $\epsilon>0$.

$f$ is a.c.
$\implies \exists \delta>0$ s.t. for any $\{(a_k, b_k)\}$ with $\sum_{k} |b_k-a_k| < \delta$, we have $\sum_{k} |f(b_k)-f(a_k)|<\epsilon$

$|Z|=0\implies \exists I=\{[\ell_i, r_i]\}_{i=1}^\infty$ cover $Z$ with $\sum_i (r_i-\ell_i)<\delta$

Let $m_i, M_i\in [\ell_i, r_i]$ s.t. 
- $f(m_i)=\inf_{[\ell_i, r_i]}f$
- $f(M_i)=\sup_{[\ell_i, r_i]} f$

both exists because $[\ell_i, r_i]$ closed.

then $f([\ell_i, r_i]) \subseteq [f(m_i), f(M_i)]$

$\implies f(Z) \subseteq f(\cup_i [\ell_i, r_i])\subseteq \cup_i [f(m_i), f(M_i)]$

Besides, $|\cup_i [f(m_i), f(M_i)]|\le \sum_i (f(M_i) - f(m_i)) < \epsilon$ from $\sum_i |M_i-m_i| \le \sum_i (r_i-\ell_i)\le \delta$

$\implies |f(Z)| \le \epsilon\forall \epsilon>0$

$\implies |f(Z)|=0$

# EX3.
$\phi$ is convex
$\iff \forall u<v$ in $(a,b)$ we have
$\phi(\frac{a+b}2) \le \frac{\phi(u) + \phi(v)}2$

$\iff \int_a^{\frac{u+v}2} fdx \le \frac 12[\int_a^u fdx + \int_a^v f dx]$

$\iff 0 \le \int_a^u f dx - 2\int_a^{\frac{u+v}2} f dx + \int_a^v f dx$
$\iff 0 \le \int_{\frac{u+v}2}^v f dx - \int_u^{\frac{u+v}2} f dx$
$\iff 0 \le \int_0^{\frac{v-u}2} f(t+\frac{u+v}2) - f(t+u)dx$

$f$ monotone increasing $\implies f(t+\frac{u+v}2) - f(t+u) \ge 0 \implies$ the inequality always hold

$\implies \phi$ is convex
